# Netflix EDA Analysis

## 📊 Project Overview
Exploratory Data Analysis of Netflix dataset to understand content distribution, ratings, and trends using Python.

**Tools:** Python, Pandas, NumPy, Matplotlib

## 🎯 What I Did
- Performed exploratory data analysis on Netflix dataset
- Analyzed content distribution across genres and countries
- Examined ratings and release trends over time
- Created visualizations to show patterns and insights
- Identified key trends in Netflix content

## 📈 Key Findings
1. **Content Distribution:** Analyzed spread across different genres
2. **Rating Patterns:** Identified most common content ratings
3. **Regional Trends:** Discovered content preferences by region
4. **Release Trends:** Tracked how Netflix content has evolved over time
