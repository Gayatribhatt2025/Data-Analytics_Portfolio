# Mental Health Analysis

## 📊 Project Overview
Analysis of mental health dataset exploring relationships between academic pressure, work pressure, job satisfaction and depression using Python.

**Dataset:** 140,700 records | **Tools:** Python, Pandas, NumPy, Matplotlib

## 🎯 What I Did
- Cleaned and explored 140,700 mental health records
- Analyzed relationships between stress factors (academic pressure, work pressure) and depression
- Created visualizations to show patterns and correlations
- Identified key factors affecting mental health

## 📈 Key Findings
1. **Academic Pressure Impact:** Significantly affects depression levels
2. **Work Pressure:** Correlates with mental health issues
3. **Job Satisfaction:** Inversely related to depression rates
4. **Age & Gender:** Show different patterns in mental health factors
