# Hospital Patient Experience & Operational Performance Dashboard

## 📊 Project Overview
Interactive Power BI dashboard analyzing patient satisfaction, wait times, and demographic patterns to improve hospital operational performance.

**Metrics Tracked:** 9,216 patients | Average wait time: 35.3 min | Satisfaction rate: 10%

**Tools:** Power BI, Data Analysis, DAX

## 🎯 What I Did
- Created comprehensive Power BI dashboard from hospital data
- Analyzed patient satisfaction across departments, race, and gender
- Tracked monthly patient volume trends
- Visualized wait time impact on satisfaction
- Built interactive filters for department and demographic analysis

## 📈 Key Metrics & Findings
1. **Total Patients:** 9,216
2. **Average Patient Age:** 39.9 years
3. **Average Wait Time:** 35.3 minutes
4. **Patient Satisfaction:** 10.0% satisfaction rate
5. **Department Performance:** Analyzed across multiple departments
6. **Demographic Insights:** Patient distribution by race and gender patterns
7. **Wait Time Correlation:** Satisfaction vs wait time analysis
